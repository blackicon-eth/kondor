# Kondor Frontend

Next.js application (App Router) serving the Kondor web interface. Users register ENS subdomains and configure automatic swap/stake/redirect rules for incoming crypto.

## Tech Stack

- Next.js (App Router, TypeScript)
- Drizzle ORM (SQLite via better-sqlite3)
- Zod (environment validation)

## Getting Started

```bash
npm install
npm run dev
```
